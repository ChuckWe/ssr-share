# ssr-share
ssr安装脚本以及bbr加速
###bbr加速脚本
```shell
wget -N --no-check-certificate "https://raw.githubusercontent.com/ChuckWe/ssr-share/master/tcp.sh"
chmod +x tcp.sh
./tcp.sh
```
###ssr安装服务脚本
```shell
wget -N --no-check-certificate "https://raw.githubusercontent.com/ChuckWe/ssr-share/master/ssr.sh"
chmod +x ssr.sh
./ssr.sh
```
